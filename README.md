# Quiz App Capstone Project

## Overview

A simple **Quiz App** that lets users take multiple-choice quizzes, see their scores, and review correct answers. Built to demonstrate React.js skills and interactive UI design.

## Features

- Take quizzes and answer questions.
- View score summary at the end.
- Option to restart the quiz.

## Technologies

- React.js
- JavaScript, JSX
- CSS

## How to Run

1. Clone the repo:

   ```bash
   git clone https://github.com/yourusername/quiz-app.git
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the app:

   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
